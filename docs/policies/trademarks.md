This project is **not affiliated with, endorsed by, or sponsored by Brother Industries, Ltd.** It is an independent open-source project that interoperates with Brother label printers via documented protocols (Brother Raster Command Reference, IEEE 802.3, RFC 3805 SNMP Printer-MIB, RFC 8011 IPP).

## Permitted use

Brother trademarks may be used in this project **only** for the descriptive purpose of stating compatibility:

- ✅ "Compatible with Brother PT-P750W"
- ✅ "Implements the Brother Raster Command Reference protocol"
- ✅ "Auto-detects PT-Series and QL-Series printers via SNMP"

## Forbidden use

- ❌ Any wording that implies endorsement, partnership, sponsorship, or affiliation with Brother Industries, Ltd.
- ❌ Use of the Brother corporate logo or distinctive brand assets in this project's icons, banners, or UI
- ❌ Distribution of Brother proprietary firmware, drivers, or copyrighted code
- ❌ Use of "Brother" in any way that could mislead users into thinking this is an official Brother product or service
- ❌ Domain names that include Brother trademarks (e.g. `brother-hub.example.com` is not permitted; use a generic name)
